# Weightlifting Exercises Reference

A structured reference of common weightlifting exercises organized by muscle group.
Includes compound (multi-muscle) and isolation (single-muscle) movements.

---

## Fundamental Movement Patterns

Every complete program should include at least one exercise from each:

| Pattern | Description | Examples |
|---------|-------------|---------|
| **Push** | Push weight away from body | Bench Press, Overhead Press, Push-Ups |
| **Pull** | Pull weight toward body | Pull-Ups, Rows, Lat Pulldown |
| **Hinge** | Hip-dominant bend | Deadlift, Romanian Deadlift, Kettlebell Swing |
| **Squat** | Knee-dominant bend | Back Squat, Goblet Squat, Leg Press |

## Compound "Big Lifts" — Foundation of Most Programs

These exercises recruit the most muscle mass and should anchor most routines:

| Lift | Primary Muscles |
|------|----------------|
| Back Squat | Quads, Glutes, Hamstrings, Core |
| Deadlift | Hamstrings, Glutes, Lower Back, Traps |
| Bench Press | Chest, Triceps, Front Delts |
| Overhead Press | Shoulders, Triceps, Upper Chest |
| Barbell Row | Lats, Rhomboids, Biceps, Rear Delts |
| Pull-Up / Chin-Up | Lats, Biceps, Core |

---

## Notes for Claude Code

- Each exercise entry includes: name, muscle group, movement type (compound/isolation)
- Equipment type can be inferred from exercise name or equipment tag
- "Main lifts" = compound, multi-joint; suitable for primary sets
- "Accessory/Isolation" = single-joint; suitable for supplemental work
- Rep/set guidelines:
  - Strength: 3–6 sets × 3–6 reps (heavy, 2–4 min rest)
  - Hypertrophy: 3–4 sets × 6–12 reps (moderate, 60–90 sec rest)
  - Endurance: 2–4 sets × 12–20+ reps (light, short rest)
